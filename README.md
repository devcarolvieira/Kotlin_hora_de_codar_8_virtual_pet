🐾 Virtual Pet Simulator
Simulador interativo de animal de estimação desenvolvido em Kotlin para prática de Programação Orientada a Objetos (POO).

Objetivo: Manter o pet equilibrado e saudável até atingir 50 anos de idade.

Funcionalidades
O jogador deve gerenciar 5 medidores críticos através de ações no console:

Alimentar: Reduz fome | Aumenta banheiro.

Brincar: Aumenta felicidade | Aumenta cansaço e sujeira.

Descansar: Recupera energia (totalmente com 8h de sono).

Banho / Banheiro: Zera os respectivos níveis de sujeira e necessidade.

Status: Exibe todos os níveis em tempo real.

Condições de Derrota
O jogo encerra se qualquer limite for atingido:

Fome, Cansaço, Sujeira ou Banheiro ≥ 100.

Felicidade ≤ 0.

Tecnologias
Linguagem: Kotlin

Paradigma: Orientação a Objetos

Como rodar: 

Copie o código do arquivo .kt.
Execute em qualquer IDE (IntelliJ IDEA) ou compilador online de Kotlin.
